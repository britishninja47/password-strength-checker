# Password Strength Checker

A simple web project that helps you check whether your password is strong enough to reduce the risk of being stolen or hacked.

## What This Project Does

- Lets you enter a password in a clean input box
- Provides a space to show password-strength feedback
- Uses a modern glass-style background and centered card UI

## Why It Matters

Weak passwords are easier to crack with brute-force attacks, leaked credential lists, and common guessing patterns. This checker is meant to encourage safer password choices before using them on real accounts.

## Tech Used

- HTML
- CSS
- Bootstrap bundle script (CDN)

## Project Structure

- index.html: Main page and password input layout
- styles.css: Visual styling, glassmorphism background, and component styles
- right-arrow.png: Button icon image

## How to Run

1. Open index.html in your browser.
2. Type a password in the input field.
3. Add or connect JavaScript logic for strength scoring if you want live validation.
